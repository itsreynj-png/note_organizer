# Note Organizer

## Project Description

Note Organizer is a Django-based web application that helps students organize their academic notes.

Users can:
- Register and Login
- Manage Courses
- Create, Edit and Delete Notes
- Search Notes by Title or Content

---

## Technologies Used

- Python 3.13
- Django
- SQLite
- Docker
- Git & GitHub

---

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Enter the project folder

```bash
cd note_management
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Apply migrations

```bash
python manage.py migrate
```

5. Run the server

```bash
python manage.py runserver
```

---

## Docker

Build and run the project:

```bash
docker compose up --build
```
